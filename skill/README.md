# TEREO Skill

The skill is the smallest TEREO rulebook:

1. make one promise
2. run one check
3. keep only what earns a receipt

Public shape:

- `promise`
- `check`
- `receipt`

Public verbs:

- `prove`
- `show`
- `log`

Its law is simple:

Gain has to beat noise.

AI may propose the change.
AI does not decide whether the proof is good enough.

## Files

- `SKILL.md`
  The main portable rulebook.
- `promise-template.md`
  A tiny template for writing a good promise.

## Rule

If the skill starts sounding like a framework manual, it is getting worse.
